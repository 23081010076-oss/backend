## 1. REGISTER & LOGIN

```bash

  -H "Content-Type: application/json" \
  -d '{

    "password": "password123",
    "password_confirmation": "password123",
    "role": "student",
    "phone": "08123456789",
    "gender": "male",
    "birth_date": "2000-01-15"
```

## 2. PROFILE MANAGEMENT

### Get My Profile
```bash
curl -X GET http://localhost:8000/api/auth/profile \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

### Update Profile
```bash
curl -X PUT http://localhost:8000/api/auth/profile \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Updated",
    "phone": "08987654321",
    "bio": "I am a student interested in programming",
    "institution": "State University",
    "major": "Computer Science",
    "education_level": "bachelor"
  }'
```

### Get Full Portfolio
```bash
curl -X GET http://localhost:8000/api/auth/portfolio \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

### Get Activity History

```bash
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

### Get Course Detail


### Create Course (Admin)
```bash

  -H "Authorization: Bearer ADMIN_TOKEN" \

    "description": "Learn HTML, CSS, and JavaScript basics",

    "level": "beginner",
    "duration": "4 weeks",
    "price": 99000,
    "access_type": "regular",
    "video_url": "https://example.com/video",

```

## 4. SCHOLARSHIPS

### List Scholarships (Public)
```bash
curl -X GET "http://localhost:8000/api/scholarships?status=open&location=Jakarta&study_field=Technology"
```

### Get Scholarship Detail
```bash
curl -X GET http://localhost:8000/api/scholarships/1
```

### Create Scholarship (Corporate)
```bash
curl -X POST http://localhost:8000/api/scholarships \
  -H "Authorization: Bearer CORPORATE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "organization_id": 1,
    "name": "Tech Innovation Scholarship 2025",
    "description": "Full scholarship for technology students",
    "benefit": "Tuition coverage 100% + mentoring",
    "location": "Jakarta",
    "status": "open",
    "deadline": "2025-12-31",
    "study_field": "Computer Science",
    "funding_amount": 50000000,
    "requirements": "GPA 3.5+, English proficiency"
  }'
```

### Apply for Scholarship (with file upload)
```bash
curl -X POST http://localhost:8000/api/scholarships/1/apply \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -F "motivation_letter=@/path/to/motivation.pdf" \
  -F "cv_path=@/path/to/cv.pdf" \
  -F "transcript_path=@/path/to/transcript.pdf"
```

### Get My Applications
```bash
curl -X GET http://localhost:8000/api/scholarships/my-applications \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

## üìù NOTES

1. Replace `YOUR_TOKEN_HERE` dengan token dari login
2. Replace `ADMIN_TOKEN`, `CORPORATE_TOKEN`, `MENTOR_TOKEN` dengan token user yang sesuai role
3. Untuk file upload, gunakan `-F` flag dengan path file
4. Path file di Windows gunakan forward slash: `C:/path/to/file.pdf`
5. Semua response adalah JSON format
