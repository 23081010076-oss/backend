### 3Ô∏è‚É£ COURSES & E-LEARNING

**List All Courses (Public):**
```
GET http://localhost:8000/api/courses?type=regular&level=beginner&access_type=free

Query Parameters:
- type: regular | bootcamp
- level: beginner | intermediate | advanced
- access_type: free | regular | premium
- search: keyword
```

**Get Course Detail:**
```
GET http://localhost:8000/api/courses/1
```

**Enroll in Course:**
```
POST http://localhost:8000/api/enrollments/1/enroll
Authorization: Bearer YOUR_TOKEN_HERE
Content-Type: application/json

{}

Status Check:
- Checks if user has required subscription
- Prevents double enrollment
- Creates enrollment with progress=0
```

**Get My Courses:**
```
GET http://localhost:8000/api/enrollments/my-courses
Authorization: Bearer YOUR_TOKEN_HERE

Returns user's enrolled courses with progress
```

**Update Progress:**
```
PUT http://localhost:8000/api/enrollments/1/progress
Authorization: Bearer YOUR_TOKEN_HERE
Content-Type: application/json

{
  "progress": 50
}

Range: 0-100
When progress=100 ‚Üí can mark as completed
```

### 4Ô∏è‚É£ SCHOLARSHIPS

**List Scholarships (Public, with Filters):**
```
GET http://localhost:8000/api/scholarships?status=open&location=Jakarta&study_field=Technology

Query Parameters:
- status: open | coming_soon | closed
- location: Jakarta, Bandung, etc
- study_field: Technology, Medicine, etc
- search: keyword
```

**Get Scholarship Detail:**
```
GET http://localhost:8000/api/scholarships/1

Returns:
{
  "id": 1,
  "organization": {...},
  "name": "Tech Scholarship",
  "description": "...",
  "benefit": "...",
  "location": "Jakarta",
  "status": "open",
  "deadline": "2025-12-31",
  "study_field": "Computer Science",
  "funding_amount": 50000000,
  "requirements": "..."
}
```

**Apply for Scholarship:**
```
POST http://localhost:8000/api/scholarships/1/apply
Authorization: Bearer YOUR_TOKEN_HERE
Content-Type: multipart/form-data

Form Data:
- motivation_letter (file: PDF/DOC, max 2MB)
- cv_path (file: PDF, max 2MB)
- transcript_path (file: PDF, max 2MB)
- recommendation_path (file: PDF/DOC, max 2MB)

Status becomes "submitted" automatically
```

**Get My Applications:**
```
GET http://localhost:8000/api/scholarships/my-applications
Authorization: Bearer YOUR_TOKEN_HERE

Returns:
[
  {
    "id": 1,
    "scholarship": {...},
    "status": "submitted",
    "submitted_at": "2025-11-13 10:30:00"
  },
  ...
]
```
